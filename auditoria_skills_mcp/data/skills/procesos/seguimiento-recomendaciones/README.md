# SKILL · seguimiento-recomendaciones

> **Tipo:** Proceso (transversal) · **Categoría:** Post-informe

Monitorear la implementación de recomendaciones de auditoría, verificar el cierre efectivo de hallazgos y reportar el estado al Comité de Auditoría.

## Cuándo se activa

Cuando el usuario menciona: seguimiento, monitoreo de hallazgos, follow-up, status de recomendaciones, cierre de hallazgo, verificación de implementación, plan de acción, hallazgo abierto/cerrado, hallazgo repetitivo, dashboard de auditoría, o al evaluar el progreso post-informe.

## Marcos de referencia anclados

- Normas Globales de Auditoría Interna (IIA) — Dominio V
- IIA Practice Guide: Monitoring and Communicating Results
