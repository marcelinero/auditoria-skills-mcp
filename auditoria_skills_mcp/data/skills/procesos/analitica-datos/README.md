# SKILL · analitica-datos

> **Tipo:** Proceso (transversal) · **Categoría:** Ejecución / CAATs

Aplicar técnicas de análisis de datos en pruebas de auditoría — ACL, IDEA, SQL, Python, R — para procesar poblaciones completas, detectar anomalías, ejecutar reglas de negocio y producir evidencia objetiva.

## Cuándo se activa

Cuando el usuario menciona: analítica, data analytics, ACL, IDEA, scripting de auditoría, prueba al 100% de la población, queries, cruces, anomalías, outliers, reglas de detección, ETL, pruebas masivas, CAATs (Computer Assisted Audit Techniques), big data en auditoría.
