# SKILL · planeacion-basada-riesgos

> **Tipo:** Proceso (transversal) · **Categoría:** Planeación

Construir el universo auditable, evaluar y priorizar riesgos, y elaborar tanto el plan anual de auditoría como los planes de trabajo individuales para cada engagement.

## Cuándo se activa

Cuando el usuario menciona: plan de auditoría, plan anual, universo auditable, evaluación de riesgos para auditoría, priorización de auditorías, alcance de auditoría, planeación de un trabajo de auditoría, memorando de planeación, o al iniciar cualquier engagement de auditoría interna.

## Marcos de referencia anclados

- Normas Globales de Auditoría Interna (IIA, 2025)
- COSO ERM 2017 · ISO 31000

## Se combina típicamente con

`evaluacion-controles` · `muestreo` · `analitica-datos` · cualquier SKILL de especialidad
