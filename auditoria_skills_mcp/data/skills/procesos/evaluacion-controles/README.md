# SKILL · evaluacion-controles

> **Tipo:** Proceso (transversal) · **Categoría:** Ejecución

Identificar, documentar, evaluar el diseño y probar la efectividad operativa de controles internos en cualquier proceso o sistema.

## Cuándo se activa

Cuando el usuario menciona: controles internos, matriz de riesgos y controles, COSO, walkthrough, pruebas de diseño, pruebas de operación, controles clave, control compensatorio, certificación SOX 404, o al auditar cualquier proceso para concluir si está adecuadamente controlado.

## Marcos de referencia anclados

- COSO Internal Control – Integrated Framework (2013)
- Modelo de las Tres Líneas (IIA, 2020)
- SOX 404 / PCAOB AS 2201

`planeacion-basada-riesgos` · `muestreo` · `papeles-trabajo` · `comunicacion-hallazgos`
