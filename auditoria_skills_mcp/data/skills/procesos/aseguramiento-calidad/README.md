# SKILL · aseguramiento-calidad

> **Tipo:** Proceso (transversal) · **Categoría:** Calidad de la función

Diseñar, operar y mantener el Programa de Aseguramiento y Mejora de la Calidad (QAIP) de la función de auditoría interna conforme al IIA, ejecutar evaluaciones internas y externas, y reportar sobre la conformidad con las Normas.

## Cuándo se activa

Cuando el usuario menciona: QAIP, EQA, External Quality Assessment, evaluación de calidad, conformidad con normas IIA, supervisión, revisión por pares, peer review, autoevaluación, KPIs auditoría, madurez de la función, encuesta de satisfacción del auditado.

## Marcos de referencia anclados

- Normas Globales de Auditoría Interna (IIA) — Dominio II
- IIA Quality Assessment Manual

## Se combina típicamente con

Opera de forma transversal y periódica sobre **toda** la función de auditoría.
