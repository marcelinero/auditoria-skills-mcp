# SKILL · comunicacion-hallazgos

> **Tipo:** Proceso (transversal) · **Categoría:** Comunicación

Estructurar hallazgos de auditoría usando los cinco atributos (CCCRA — Condición, Criterio, Causa, Consecuencia, Acción), redactar informes claros y persuasivos, y comunicar resultados a la administración, comité de auditoría y otras partes interesadas.

## Cuándo se activa

Cuando el usuario menciona: hallazgo, observación, informe de auditoría, draft de informe, recomendación, plan de acción, severidad, ranking de hallazgos, calificación, conclusión, opinión, o al redactar la salida formal del trabajo de auditoría.

- IIA Practice Guide: Audit Reports
