# SKILL · auditoria-operativa

> **Tipo:** Especialidad · **Dominio:** Procesos y eficiencia

Evaluar la eficiencia, eficacia y economía de procesos operativos, identificar oportunidades de mejora y verificar el logro de objetivos del proceso.

## Cuándo se activa

Cuando el usuario menciona: auditoría operativa, auditoría de procesos, eficiencia operacional, productividad, eficacia, value for money, mejora de procesos, mapeo de procesos, KPIs operativos, BPM, lean, six sigma en auditoría, performance audit, throughput, ciclo de proceso.

## Marcos de referencia anclados

- IIA Standards · ISO 9001
- INTOSAI ISSAI 3000 (auditoría de desempeño)
