# SKILL · auditoria-cumplimiento

> **Tipo:** Especialidad · **Dominio:** Cumplimiento legal y regulatorio

Evaluar la adherencia de la organización a leyes, regulaciones, normativas internas y compromisos contractuales aplicables, así como la efectividad del programa de cumplimiento.

## Cuándo se activa

Cuando el usuario menciona: auditoría de cumplimiento, compliance audit, regulatory audit, programa de cumplimiento, ISO 37301, ISO 37001, antisoborno, anticorrupción, FCPA, UK Bribery Act, SAGRILAFT, SARLAFT, AML/CFT, lavado de activos, código de ética, conflicto de interés, sanciones, due diligence de terceros, oficial de cumplimiento.

## Marcos de referencia anclados

- ISO 37301 (compliance) · ISO 37001 (antisoborno)
- OCDE — Convención Anticohecho · marcos sectoriales
