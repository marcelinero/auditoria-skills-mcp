# SKILL · auditoria-calidad

> **Tipo:** Especialidad · **Dominio:** Sistemas de gestión de calidad

Auditar sistemas de gestión de calidad conforme a ISO 9001 e ISO 9004, aplicando las directrices de ISO 19011 para auditorías de sistemas de gestión, evaluar el cumplimiento de requisitos del cliente y la mejora continua.

## Cuándo se activa

Cuando el usuario menciona: auditoría de calidad, ISO 9001, ISO 9004, ISO 19011, sistema de gestión de calidad, SGC, mejora continua, no conformidad, acción correctiva, acción preventiva, satisfacción del cliente, política de calidad, manual de calidad, KPI calidad, certificación ISO.

- ISO 19011 (auditorías de sistemas de gestión)
