# SKILL · auditoria-financiera

> **Tipo:** Especialidad · **Dominio:** Información financiera

Realizar auditorías sobre estados financieros, cuentas, ciclos y aseveraciones contables, evaluando si están preparados conforme a NIIF/IFRS u otro marco aplicable y si los controles sobre la información financiera son efectivos.

## Cuándo se activa

Cuando el usuario menciona: auditoría financiera, estados financieros, ciclo contable, aseveraciones, NIA, ISA, NIIF, IFRS, US GAAP, materialidad financiera, cierre contable, conciliación, SOX 404, ICFR, opinión de auditor, o al auditar cualquier cuenta o saldo contable.

## Marcos de referencia anclados

- NIA / ISA (IAASB)
- NIIF / IFRS · US GAAP
- PCAOB · SOX 302 y 404 · COSO IC-IF

## Se combina típicamente con

`planeacion-basada-riesgos` · `evaluacion-controles` · `muestreo` · `analitica-datos` · `papeles-trabajo` · `comunicacion-hallazgos`
