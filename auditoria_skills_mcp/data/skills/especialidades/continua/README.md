# SKILL · auditoria-continua

> **Tipo:** Especialidad · **Dominio:** Auditoría continua / monitoreo continuo

Diseñar, implementar y operar capacidades de auditoría continua y monitoreo continuo, con ejecución automatizada y recurrente de procedimientos analíticos sobre poblaciones completas para detección temprana y cobertura amplia.

## Cuándo se activa

Cuando el usuario menciona: auditoría continua, continuous auditing, continuous monitoring, monitoreo continuo, GTAG 3, automatización de pruebas, dashboard de auditoría, alertas de auditoría, KRI, key risk indicators, real-time auditing, near-real-time, scripts recurrentes, plataforma analítica de auditoría.
