# SKILL · auditoria-gestion-desempeno

> **Tipo:** Especialidad · **Dominio:** Desempeño organizacional y gobierno

Evaluar a nivel agregado el logro de objetivos estratégicos y desempeño organizacional, la efectividad del gobierno corporativo, la gestión de riesgos empresariales y la creación de valor sostenido.

Cuando el usuario menciona: auditoría de gestión, performance audit, auditoría de desempeño, gobierno corporativo, ERM, COSO ERM, ISSAI, INTOSAI, balanced scorecard, OKR, KPI estratégico, junta directiva, comité de auditoría, OCDE governance, ISO 37000, accountability, value for money.
